본 매뉴얼을 읽기 전에, 카이저랩 AgentQC 프로그램이 어떤 구조로 동작하는지,
그리고 정상적인 QC 데이터 흐름을 위해 반드시 이해해야 할 기본 개념을 설명할 것이에요!

정도관리 데이터는 이렇게 들어와요

정도관리 프로그램은 장비와 직접 연결되어 있지 않아요.
반드시 인터페이스 프로그램을 거쳐서 데이터가 들어와요. 따라서, 인터페이스 프로그램이 켜져 있어야 정도관리 프로그램에 데이터가 정상적으로 들어올 수 있어요.

정도관리 프로그램은 보조 도구예요

정도관리 결과를 규칙에 맞게 분석해 아래와 같이 상태를 알려줘요.

정상: 현재 검사 상태가 안정적입니다.
주의: 바로 중단할 필요는 없지만, 확인이 필요합니다.
거부: 조치 없이 검사를 진행하기 어렵습니다.

해당 판단은 도움용 안내에요. 최종 판단과 조치는 항상 사용자가 결정해야 해요.

숫자만 중요한 게 아니에요

정도관리 프로그램은 결과값뿐 아니라 어떤 조건에서 측정됐는지 함께 관리해요.

  • 정도관리 물질 LOT(Target값)
  • 검사항목, 레벨
  • 장비 정보
  • 측정 날짜, LOT의 시작일자, 종료일자, 유효일자 등

이 정보가 정확해야 나중에 문제가 생겼을 때 원인을 빠르게 찾을 수 있어요.
마스터 코드 관리를 성실하게 진행해주세요!

이것만 기억하면 충분합니다 (최종 3줄 요약)

정도관리 데이터는 인터페이스 프로그램이 켜져 있어야 들어옵니다. 실행 중인지 확인해주세요.

정도관리 프로그램에서 나온 결과 해석의 참고용이며, 최종 판단 책임은 사용자에게 있어요.

정확한 LOT 정보가 정도관리의 기본이에요. 마스터 코드 관리에 성실히 임해주세요.

QC 결과 확인 (홈 > QC 결과 확인)

기능 개요

QC 결과 확인은 특정 장비·물질의 상세 결과를 Levey-Jennings 차트와 함께 확인하는 화면이에요.
테이블에서 결과를 선택하면 LJ 차트에 위치가 연동되고, 이상치에 Comment 등록이나 결과 삭제도 할 수 있어요. 조회 결과는 Excel로 저장하거나 차트·테이블을 함께 인쇄할 수 있어요.

화면 구성

위 — 조회 조건

검색 조건 필터
데이터 조회결과 테이블에 표시할 기간 (시작일자 ~ 종료일자)
차트 조회Levey-Jennings 차트에 표시할 기간 (데이터 조회 기간과 독립 설정 가능)
검사실/장비/물질검사실/장비/물질선택
물질구분QC / 허용범위 설정용 등
검사항목/레벨전체 또는 특정 검사항목/레벨 선택

중앙 — Levey-Jennings 차트

구성 요소
범례 (L1 / L2 / L3)LOT별 라인 색상 구분. LOT 코드와 유효기간 표시
±3SD 기준선빨간 수평선 — 이 선을 넘으면 1(3s) Rule 위반
Mean 기준선검정 수평선 — 설정된 타깃 평균값
데이터 포인트LOT별 색상으로 구분된 QC 결과값 점/라인
빨간 수직선Rule 위반 발생 시점 표시
초록 수직선선택된 데이터 포인트 위치 표시

하단 — 검사항목별 결과 테이블

Level 1 / Level 2 / Level 3를 가로로 나란히 표시하며, 동일 순번의 결과를 한 행에서 비교할 수 있어요.

컬럼
검사항목/검사일시검사 순번/결과가 수신된 날짜 및 시간
결과측정 결과값. 선택 시 차트에 초록 수직선으로 위치 표시됨
Rule위반된 Westgard Rule 표시 (예: 1(3s))
Range해당 LOT의 허용범위 (하한~상한)
Comment등록된 코멘트 표시

작업 순서

① 검사실 · 장비 · 물질 · 기간 선택 후 조회
② LJ 차트에서 이상 구간(빨간 수직선) 확인
③ 결과 테이블에서 해당 행 클릭 → 차트와 위치 연동 확인
④ 필요 시 Comment 등록 / 결과 삭제 / Excel 출력 진행

주의사항

데이터 조회와 차트 조회 기간은 각각 독립적으로 설정할 수 있어요.
(예: 차트는 최근 1개월, 테이블은 최근 1주일만 표시)

Target 재조정이 Comment 열에 표시된 경우, 해당 시점에 LOT Target값이 변경되었음을 의미해요.

Rule 위반이 의심될 경우 해당 시점 전후 데이터를 함께 확인해주세요.

대시보드 (홈 > 대시보드)

기능 개요

정도관리 결과를 상단 요약 지표 + 레벨별 현황 + 상세 테이블 + LJ 차트로 한 번에 확인하는 메인 화면이에요.
레벨별·검사별 보기를 전환하거나 일별·주별·월별 단위로 기간을 바꿔 단기 이상과 장기 추세를 함께 점검할 수 있어요. 도넛 차트나 LJ 차트 범례로 특정 LOT·레벨만 선택해 집중적으로 보거나, Multi Sort로 이상 데이터를 빠르게 추적할 수 있어요.

화면 구성

좌측 — 장비 목록

장비 선택 패널
장비 썸네일이미지와 장비명으로 나열. 클릭 시 해당 장비 데이터로 전환
상단 탭검사실 단위 탭 구성 (예: 혈액학). 해당 검사실 소속 장비만 표시

상단 — 옵션 바/요약 카드

조회 조건 설정
보기 옵션레벨별 보기 / 검사별 보기 선택. 검사별 보기 전환 시 검사항목 탭 중심으로 변경
기간 단위일별 / 주별 / 월별 집계 단위 선택
날짜 구간시작일 ~ 종료일 직접 지정
물질 종류ALL 또는 특정 물질 종류 선택으로 결과 필터링
총 검사건수조회 기간 내 해당 검사실의 전체 QC 결과 건수
정상/주의/거부기간 내 검사실의 정상/주의/거부 건수

상단 우측 — 통계 정보 박스

4개 통계 정보 영역
Rule 설정 정보적용 중인 Westgard Rule 목록과 각 Rule 판정 결과(정상/거부) 표시
Target 설정 정보선택 LOT의 타깃 Mean, Sd, Cv 값
Period statistics조회 기간 내 실측 통계 — LOT No, 결과개수, Mean, Sd, Cv
CumulativeLOT 전체 사용기간 누적 통계 — LOT No, 결과개수, Mean, Sd, Cv

중앙 — 레벨 탭 & Level 카드

레벨별 현황
ALL / 1 / 2 / 3 탭전체 또는 특정 레벨 선택 — 해당 레벨 카드만 표시
Level 카드LOT 번호, 유효기간, 장비명, 총 건수 표시. 사용 중 LOT는 별도 카드로 구분
도넛 차트정상·주의·거부 비율 시각화. 거부 발생 시 빨간 조각 표시
상태 건수카드 내 정상 주의 거부 건수 직접 확인

하단 좌측 — 결과 테이블

결과 테이블
Multi Sort체크 후 열 제목 클릭 — 오름차순 → 내림차순 → 초기화
행 선택행 클릭 시 우측 LJ 차트에서 해당 포인트 위치 강조 표시

하단 우측 — Levey-Jennings Chart

LJ 차트
차트 제목장비명 · 검사항목명 표시 (예: BC-6D - WBC - Levey-Jennings Chart)
범례LOT별 Lv1·Lv2·Lv3 색상 구분. 사용 중 LOT와 종료 LOT는 별도 색
기준선Mean, ±1SD~±4SD 수평선 표시 — 포인트 위치 판단 기준
범례 on/off범례 클릭으로 특정 LOT·레벨 표시/숨김

작업 순서

① 장비를 선택하고 보기 옵션, 기간 단위, 날짜 구간을 먼저 설정해요.
② 지정된 기간 내 검사 결과 테이블과 LJ차트를 확인해요. 필요 시, 검사별 보기로 바꿔 검사항목 중심으로 봐요.
③ 거부 박스나 이상 결과 행을 기준으로 하단 결과 테이블에서 Rule, Range, Comment를 확인해요.
④ 추가로 Comment를 바로 등록하거나 상세 화면으로 이동해 등록하고,
오른쪽 LJ 차트에서는 범례 on/off로 비교 대상을 좁혀서 판단해요.

주의사항

상단의 레벨별 보기 / 검사별 보기는 화면 해석 방식이 달라지므로, 현재 어떤 보기인지 먼저 확인해주세요.

거부 건수만 보지 말고 Period statistics, Cumulative, LJ 차트를 같이 봐야 일시적 이상인지 지속적 이동인지 판단하기 쉬워요.

Comment에 Target 재조정 같은 이력이 있으면 차트 변동을 해석할 때 반드시 함께 고려해주세요.

데이터 삭제 기능은 QC 신뢰성을 위해 병원 요청이 있을 경우에만 해당 사용자에게 권한을 부여해요.

바코드 출력 (홈 > 바코드 출력)

기능 개요

QC 물질을 선택해 접수용 바코드를 생성하고 출력하는 화면이에요.
접수와 동시에 인쇄하거나, 하단 이력 목록에서 재출력·접수 취소를 진행할 수 있어요. 라벨 위치·크기 등 출력 설정을 저장해두면 다음 출력에도 그대로 적용할 수 있어요.

화면 구성

위 — 접수 조건 입력

바코드 생성 조건
검사실/장비/물질/물질 종류/레벨검사실/장비/물질/물질 종류/레벨 선택
접수일자접수 기준 날짜 지정
접수시 Barcode 출력체크 시 접수와 동시에 바코드 인쇄
프린터 / 접수 버튼출력 프린터를 선택하고 접수 버튼으로 바코드 생성

하단 — 바코드 조회

생성된 바코드 목록을 조회해요. Barcode 번호만 체크 시 이 화면에서 생성한 바코드만 필터링해요.

중앙 - 접수 대상, 라벨 미리보기, 출력 위치 설정

영역
검사항목 선택선택한 물질에 해당하는 검사 항목 목록에서 접수할 항목을 선택
접수 대상선택된 물질명, 레벨, 검사코드, 검사명이 접수 대상 표에 표시됨
라벨 미리보기출력될 바코드 형태를 화면에서 미리 확인
바코드 출력 설정X/Y 시작 위치, 여백, 폰트 크기, 비율, 높이 등 라벨 인쇄 좌표를 조정
바코드 설정 저장조정한 라벨 출력 설정값을 저장

하단 - 접수 정보 조회 리스트

생성된 바코드 접수 이력을 목록으로 조회해요.
물질 종류와 접수 정보 기간으로 조회할 수 있고, Barcode 번호만 체크 시 바코드 번호 기준으로 빠르게 확인할 수 있어요.

컬럼
바코드번호접수 시 생성된 고유 바코드 번호
레벨접수된 레벨 정보
검사코드 / 검사명접수 대상 검사 항목 정보
결과일자접수 또는 결과 기준 일자

작업 순서

① 좌측에서 검사실과 장비를 선택하고, 상단에서 물질·물질 종류·레벨·접수일자를 지정
② 검사항목을 선택해 접수 대상 목록에 포함할 항목을 확인
③ 필요 시 '접수시 Barcode 출력' 체크와 프린터 선택, 라벨 미리보기 및 출력 설정 확인
④ 접수 버튼을 눌러 바코드를 생성하고, 하단 접수 정보 목록에서 조회·재출력·접수 취소 진행

주의사항

바코드 접수와 출력은 별개의 기능이에요.

접수 시 바로 출력하려면 반드시 '접수시 Barcode 출력'을 체크해야 해요.

체크 없이 접수한 경우 접수 정보창의 Barcode 재출력 버튼으로 다시 출력할 수 있어요.

라벨 위치가 프린터와 맞지 않으면 중앙의 바코드 출력 설정 값과 바코드 설정 저장 기능을 사용해 출력 위치를 먼저 조정해 주세요.

QC 데이터 조회 (홈 > QC 데이터 조회)

기능 개요

대시보드보다 세밀한 조건으로 QC 데이터를 조회하고 후속 처리까지 진행하는 화면이에요.
상태·규칙 등으로 필터링해 조회하고, Comment 등록 및 Rule Refresh도 이 화면에서 진행이 가능해요.

환자 결과 확인 (환자 결과 확인)

기능 개요

기능 개요

LOT FAQ

물질 종류(검사실 Target, 허용범위용 Target, 제조사 Target)가 무엇인가요?
검사실 Target: 일반 QC LOT로 사용 시 이 설정을 선택하면 돼요.
허용범위용 Target: 허용범위 설정을 위해 임시로 사용하는 LOT 용도에요.
제조사 Target: 물질 제조사에서 제공한 값으로, 일반적인 조회 용도로 사용해요.
등록된 리스트를 조회할 때 조회 옵션 ALL, LAST는 무엇을 의미하나요?
LAST: 현재 사용중인 LOT를 조회하고 싶을 때
ALL: 기존에 사용하던 LOT까지 전체 조회하고 싶을 때
BIAS(%)와 허용치가 무엇인가요?
BIAS(%): QC 결과값이 대상 평균 값에서 얼마나 치우쳐도 허용할 지를 선택 기준이에요.
허용치: Target값(mean, SD, CV)과 달리 '이 범위까지는 정상이로 보겠다.'라고 정해둔 허용 기준이에요.
장비를 선택했는데, 아무것도 나타나지 않아요
마스터 통합관리는 기존에 등록된 LOT 기준으로 표시돼요!
만약 첫 LOT 등록이라면 아무것도 보이지 않을 수 있어요.
이런 경우에는 모든 정보 보기 체크박스를 체크해주세요!
→ 모든 물질 정보 보기   → 모든 시약 정보 보기   → 모든 검사 정보 보기

장비물질ID나 장비검사CD가 무엇인지 모르겠어요
장비물질ID와 장비검사CD는 인터페이스와 통신하기 위한 약속 코드(필수값)에요.
장비검사CD는 보통 자동으로 채워지지만, 장비물질ID는 변경되는 값으로 직접 입력해야 해요.

장비물질ID: 인터페이스 첫 화면에 보이는 검체번호가 장비물질ID에요. 레벨별로 값이 다르니 확인 후 입력해주세요.
장비검사CD: 인터페이스 첫 화면 → 설정 → 검사 정보설정 → QC Match 탭에서 보이는 약속 코드에요.

입력하기 어려운 상황에는 카이저랩으로 연락해주세요
LOT NO나 장비물질ID 같은 코드는 중복이 많아서 복사 붙여넣기 하기 귀찮아요
맨 위에 중심값을 하나 입력하고, '일괄 적용' 버튼을 눌러주세요.
그러면 모든 검사항목에 동일한 값이 입력돼요.
LOT Target값을 잘못 입력했어요. 이미 등록된 LOT의 값을 수정하고 싶은데 어떻게 하죠?
수정하고 싶은 데이터를 조회 → 값을 수정하고 수정 버튼을 클릭하세요.
수정된 데이터는 저장하기 전까지 빨간색으로 보여요.
LOT 시작일자(적용기간)를 변경하고 싶어요
조회 옵션을 ALL로 변경 후 조회 → 수정/종료일자 수정 가능 모드로 변경하면 LOT 적용 기간을 변경할 수 있어요.
*LOT 적용 기간을 변경한다고 해서 적용되었던 Westgard Multi Rule 위반 사항이 변경되지는 않아요. 해당 LOT의 변경된 기간 내의 모든 데이터에 대해 'Rule 새로고침'을 진행해야 해요.

기본 마스터 관리
(프로그램 설정 > 기본 마스터 관리)

기능 개요

검사실·장비·MultiRule·Comment·단위 등 정도관리 기본 환경을 설정하는 화면이에요.
마스터 구분 드롭다운으로 설정 항목을 전환하고, 신규·수정·삭제 버튼으로 마스터 데이터를 관리할 수 있어요. 사용자 계정 설정은 별도 섹션(계정 생성 / 비밀번호 변경)에서 확인할 수 있어요.

검사실 설정 — 마스터 구분 '검사실'

입력 항목

파트장/대표번호/검사항 갯수담당자 정보/대시보드 검사항목 탭 표시 개수

Multi Rule 설정 — 마스터 구분 'Multi Rule'

검사실별 Westgard Multi Rule 적용 기준을 설정해요. 칸 클릭으로 Rule을 적용·해제할 수 있어요.

입력 항목

Comment 설정 — 마스터 구분 'Comment'

자주 사용하는 코멘트를 저장해두면 QC 결과 확인에서 빠르게 선택할 수 있어요.

단위 설정 — 마스터 구분 '단위'

특수기호가 많은 단위를 미리 등록해두면 검사항목 등록 시 선택해 사용할 수 있어요.


화면 구성

위 — 마스터 구분 선택

마스터 구분 드롭다운으로 설정 항목(검사실 / 장비 / Multi Rule / Comment / 단위)을 전환해요.

하단 — 해당 설정 입력 영역

선택한 마스터 구분에 맞는 입력 필드와 목록이 표시돼요.

작업 순서

① 마스터 구분 드롭다운에서 설정할 항목 선택
② 신규 버튼 클릭 → 필드 입력
③ 등록 버튼 클릭 (Multi Rule은 칸 클릭 후 등록)

주의사항

검사 항 갯수는 대시보드에서 검사항목 탭(Tab)을 몇 개까지 표시할지 결정해요.

검사항목이 많은 경우 이 값을 올려야 대시보드에서 모든 항목이 보여요.

장비 신규 등록 또는 수정 시에는 피어그룹 관리가 연결될 수 있으므로 반드시 카이저랩에 연락해주세요.

MultiRule 수정 후에는 반드시 등록 버튼을 클릭해야 저장돼요.

④ 우측 하단 신규버튼 클릭 후, 검사실, 사용자 아이디, 사용자명 입력 후 등록 → 재로그인
