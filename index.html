본 매뉴얼을 읽기 전에, 카이저랩 AgentQC 프로그램이 어떤 구조로 동작하는지,
그리고 정상적인 QC 데이터 흐름을 위해 반드시 이해해야 할 기본 개념을 설명할 것이에요!

정도관리 데이터는 이렇게 들어와요

정도관리 프로그램은 장비와 직접 연결되어 있지 않아요.
반드시 인터페이스 프로그램을 거쳐서 데이터가 들어와요. 따라서, 인터페이스 프로그램이 켜져 있어야 정도관리 프로그램에 데이터가 정상적으로 들어올 수 있어요.

정도관리 프로그램은 보조 도구예요

정도관리 결과를 규칙에 맞게 분석해 아래와 같이 상태를 알려줘요.

정상: 현재 검사 상태가 안정적입니다.
주의: 바로 중단할 필요는 없지만, 확인이 필요합니다.
거부: 조치 없이 검사를 진행하기 어렵습니다.

해당 판단은 도움용 안내에요. 최종 판단과 조치는 항상 사용자가 결정해야 해요.

숫자만 중요한 게 아니에요

정도관리 프로그램은 결과값뿐 아니라 어떤 조건에서 측정됐는지 함께 관리해요.

  • 정도관리 물질 LOT(타깃값)
  • 검사항목, 레벨
  • 장비 정보
  • 측정 날짜, LOT의 시작일자, 종료일자, 유효일자 등

이 정보가 정확해야 나중에 문제가 생겼을 때 원인을 빠르게 찾을 수 있어요.
마스터 코드 관리를 성실하게 진행해주세요!

이것만 기억하면 충분합니다 (최종 3줄 요약)

  • 정도관리 데이터는 인터페이스 프로그램이 켜져 있어야 들어옵니다. 실행 중인지 확인해주세요.
  • 정도관리 프로그램에서 나온 결과 해석의 참고용이며, 최종 판단 책임은 사용자에게 있어요.
  • 정확한 LOT 정보가 정도관리의 기본이에요. 마스터 코드 관리에 성실히 임해주세요.

QC 결과 확인 (메뉴 > 홈 > QC 결과 확인)

기능 개요

대시보드가 전체 현황을 한눈에 보는 화면이라면, QC 결과 확인은 특정 장비·물질의 상세 결과를 Levey-Jennings 차트와 함께 확인하는 화면이에요.
이상 결과가 발생했을 때 원인을 추적하거나, Comment 등록 및 결과 삭제 등 후속 처리를 할 때 사용해요.

차트 조회Levey-Jennings 차트에 표시할 기간 (데이터 조회 기간과 독립 설정 가능)

검사항목전체 또는 특정 검사항목 선택

Level 1 / Level 2 / Level 3를 가로로 나란히 표시하며, 동일 순번의 결과를 한 행에서 비교할 수 있어요.

인쇄차트 + 테이블을 함께 인쇄

작업 순서

① 검사실 · 장비 · 물질 · 기간 선택 후 조회
② LJ 차트에서 이상 구간(빨간 수직선) 확인
③ 결과 테이블에서 해당 행 클릭 → 차트와 위치 연동 확인
④ 필요 시 Comment 등록 / 결과 삭제 / Excel 출력 진행

주의사항

데이터 조회와 차트 조회 기간은 각각 독립적으로 설정할 수 있어요.
예: 차트는 최근 1개월, 테이블은 최근 1주일만 표시
Target 재조가 Comment 열에 표시된 경우, 해당 시점에 LOT 타깃값이 변경되었음을 의미해요.
Rule 위반이 의심될 경우 해당 시점 전후 데이터를 함께 확인해주세요.

생성된 바코드 목록을 조회해요. Barcode 번호만 체크 시 이 화면에서 생성한 바코드만 필터링해요.

주요 기능

기능설명
Comment 등록데이터 선택 → 코멘트 등록 버튼 → 코멘트 선택/입력 → 등록
Excel 출력조회한 모든 데이터를 엑셀로 저장 (선택 여부 무관)

작업 순서

② 결과 테이블에서 대상 데이터 확인
③ 필요 기능 실행 — Comment 등록 / Excel 출력 / 중복결과 취소 / Rule Refresh

Rule Refresh

① 등록된 결과가 있는 LOT의 타깃값 변경
② 해당 LOT의 시작일자, 종료일자를 데이터 조회 구간으로 설정

LOT FAQ

물질 종류(검사실 Target, 허용범위용 Target, 제조사 Target)가 무엇인가요?
검사실 Target: 일반 QC LOT로 사용 시 이 설정을 선택하면 돼요.
허용범위용 Target: 허용범위 설정을 위해 임시로 사용하는 LOT 용도에요.
제조사 Target: 물질 제조사에서 제공한 값으로, 일반적인 조회 용도로 사용해요.
등록된 리스트를 조회할 때 조회 옵션 ALL, LAST는 무엇을 의미하나요?
LAST: 현재 사용중인 LOT를 조회하고 싶을 때
ALL: 기존에 사용하던 LOT까지 전체 조회하고 싶을 때
BIAS(%)와 허용치가 무엇인가요?
BIAS(%): QC 결과값이 대상 평균 값에서 얼마나 치우쳐도 허용할 지를 선택 기준이에요.
허용치: 타깃값(mean, SD, CV)과 달리 '이 범위까지는 정상이로 보겠다.'라고 정해둔 허용 기준이에요.
장비를 선택했는데, 아무것도 나타나지 않아요
마스터 통합관리는 기존에 등록된 LOT 기준으로 표시돼요!
