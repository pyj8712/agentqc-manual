================================================================
  파일명 자동 변환 & 월별 정리 도구 사용 방법
================================================================

[ 지원 파일 종류 ]

  PDF (텍스트 선택 가능한 디지털 문서)
    - 전자세금계산서 → 년월일 세금계산서(거래처-품목).pdf
    - 거래명세서     → 년월일 거래명세서(거래처-적요).pdf

  이미지 (영수증 사진 등)
    - JPG, PNG 등   → 년월일 영수증(금액원).확장자
    ※ 이미지 처리는 Gemini API 키 필요 (최초 1회 입력 또는 .env 저장)


[ 처리 완료 후 자동 이동 경로 ]

  세금계산서 · 거래명세서
    → C:\Users\yujin\OneDrive\Desktop\박유진\1. 매출서류\YYYY-MM\

  영수증
    → C:\Users\yujin\OneDrive\Desktop\박유진\0-1. 재무관리\YYYY-MM\

  처리 실패한 파일
    → 이 폴더 안의 error\ 폴더로 이동


================================================================
  실행 방법 1 — 일괄 처리 (실행.bat)
================================================================

  1. 변환할 파일을 아래 폴더에 넣습니다.
       rename-tax-invoices\pdf\

  2. [실행.bat] 을 더블클릭합니다.

  3. 변환될 파일명과 이동 경로 미리보기가 표시됩니다.

  4. y 입력 → 이름 변경 후 해당 월별 폴더로 자동 이동
     n 입력 → 취소


================================================================
  실행 방법 2 — 자동 감시 모드 (감시_실행.bat)
================================================================

  1. [감시_실행.bat] 을 더블클릭합니다.

  2. pdf\ 폴더를 계속 감시합니다.
     파일을 넣으면 자동으로 처리 후 월별 폴더로 이동합니다.
     확인 없이 즉시 처리되므로 주의하세요.

  3. 종료하려면 창에서 Ctrl+C 를 누릅니다.


================================================================
  변환 예시

    → 1. 매출서류\2026-04\ 에 저장

    KakaoTalk_20260504_122954054.jpg
    → 20260504 영수증(15,000원).jpg
    → 0-1. 재무관리\2026-05\ 에 저장


================================================================
  Gemini API 키 설정 (이미지·영수증 처리 시 필요)
================================================================

  방법 1 — 실행 시 입력
    프로그램 실행 중 키 입력 요청이 나타나면 입력합니다.
    저장 여부를 물어보면 y 입력 시 자동 저장됩니다.

  방법 2 — .env 파일에 미리 저장
    rename-tax-invoices\.env 파일을 메모장으로 열어
    아래 형식으로 저장합니다.

      GEMINI_API_KEY=여기에_키_입력

  API 키 발급: https://aistudio.google.com


================================================================
  필요 프로그램
================================================================

  Python 3.10 이상 및 아래 라이브러리 (최초 1회만 설치)
  실행.bat / 감시_실행.bat 실행 시 자동으로 설치됩니다.

    pip install PyMuPDF google-genai watchdog
