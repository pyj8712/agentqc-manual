================================================================
  PDF 파일명 자동 변환 도구 사용 방법
================================================================

[ 지원 문서 종류 ]
  - 전자세금계산서 → 년월일 세금계산서(공급받는자 - 품목).pdf
  - 거래명세서     → 년월일 거래명세서(거래처명 적요).pdf


[ 사용 순서 ]

  1. 변환할 PDF 파일을 아래 폴더에 넣습니다.
     C:\Users\yujin\OneDrive\Desktop\claude\rename-tax-invoices\pdf

  2. 터미널(명령 프롬프트 또는 PowerShell)을 열고
     아래 명령어를 실행합니다.

     python "C:\Users\yujin\OneDrive\Desktop\claude\rename-tax-invoices\rename_tax_invoice.py"

  3. 변환될 파일명 미리보기가 화면에 표시됩니다.

  4. 내용을 확인한 후 y 를 입력하면 파일명이 변경됩니다.
     취소하려면 n 을 입력합니다.


[ 변환 예시 ]

  세금계산서
    1.PDF
    → 20260309 세금계산서((주)랩메디라인 - 인터페이스 설치 본바움병원 BC30 외 5건).pdf

  거래명세서
    0309 거래명세서(랩메디라인-본바움병원).PDF
    → 20260309 거래명세서((주)랩메디라인 본바움병원 인터페이스 설치건).pdf


[ 주의 사항 ]

  - 텍스트 선택이 되는 디지털 PDF만 지원합니다. (스캔 이미지 불가)
  - 세금계산서와 거래명세서를 같은 폴더에 섞어 넣어도 됩니다.
    스크립트가 자동으로 종류를 구분합니다.
  - 같은 이름의 파일이 이미 존재하면 건너뜁니다. (덮어쓰기 없음)
  - 변환에 실패한 파일은 건너뛰고 나머지는 정상 처리됩니다.


[ 필요 프로그램 ]

  Python 및 pymupdf 라이브러리가 설치되어 있어야 합니다.
  최초 1회만 설치하면 됩니다.

  pip install pymupdf
